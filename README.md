# Academic Slides

This is a fork of the excellent [Slides Extended](https://github.com/ebullient/obsidian-slides-extended) Obsidian plugin by Erin Schnabel which is a fork of [Advanced Slides](https://github.com/MSzturc/obsidian-advanced-slides) by  Matthäus Szturc.

The goals of this project:

- [ ] Simplify the code and make it more scalable and fully commented.
- [ ] Make it comply with Obsidian's plugin safety rules.
- [ ] Make it work better with native Obsidian syntax.
- [ ] Add CJK support.
- [ ] Add most of the functionality of [Quarto's version of reveal.js](https://quarto.org/docs/presentations/revealjs/)
- [ ] Add additional functionality useful in academical or educational context, such as timelines (See https://dvirtz.github.io/reveal-timeline/) or quizes (See https://github.com/juwit/reveal-quiz)

This is not intended to be a replacement for Slides Extended but a purpose-built version usable in academical/educational context.
(Some functionality available in Slides Extended might be even removed.)

This project is very early in development.
